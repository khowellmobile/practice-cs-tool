Generate Report

Generate report is used to create a table listing times spent on each client. Enter in the time date and range or select one of the preset time ranges. If you have a report type you use often, check the history section on the left. It keeps the 25 most recent reports.

Time Range Presets

  • Year to Date (YTD) - This preset is the time range from January 1st of the current year to the current date
  • All Time - This preset is the time range of the current date to the beginning of time!
  • Last Year - This preset is the time range of January 1st of last year to December 31st of last year
  • Custom - This is more an indicator that a custom time range has been entered. The preset automatically will change to custom when a user enters a date.

Change Database

Change Database lets you switch the web application's source database. It requires four fields, with optional username and password inputs. Entering these disables Windows authentication. The current connection is shown on the right, with status updates and errors displayed in the Status/Messages box during connection setup

Form Field Inputs

  • Database Engine - The engine for the database (mssql, sqlite, mysql, etc)
  • Database Name - The name of the database
  • Database Host - This is the host server of the database
  • Database Driver - This is the set od protocols used to control the databse
  • Username (optional) - Your username to login to the database
  • Password (optional) - Your password to login to the database

Account Information

Account Information lets you change your accounts name, username/email, and password. Each of these fields have limits as to their length and allowed characters. Check below more more information on the paramters.

Information Fields

  • Account Name - Your first and last name. Limited to only alphabet characters.
  • Email - Your email is used as both your username and email. It is limited to normal email characters and the email format.
  • Password - Your password must be at least 8 characters long, contain one special character, and one number.

Report History

Report history is an expanding side menu with a scroll set of buttons that represent the 25 most recent reports. Click on a history button to regenerate that report with the same paramters.

Each history button is split into 4 parts

  • Time Range Preset - This word or symbol represents the time range at a glance (All Time, Last Year, Etc).
  • Report Date - This is the date that the report was ran on.
  • Start Date - This is the starting date for the range of the report data.
  • End Date - This is the ending date for the range of the report data.

Symbol Meanings

  • YTD - Year to Date - Time range from January 1st of the current year to today.
  • AT - All Time - Time range from beginning of records to current date.
  • LY - Last Year - Time range spanning from January 1st of last year to December 31st of last year.
  • C - Custom - Indicates a custom time range.

Parameters/Buttons

The section at the top is used to hold parameter inputs and buttons to do functions. Listed below are the fields and buttons in order from left to right.

List of functions

  • Left Date Input - This field holds the date for the start date of your report.
  • Right Date Input - This field holds the date for the enf date of your report.
  • Preset Selector - Select a date preset. Changing either of the date inputs will set this field to custom.
  • Create Report Button - Once entering the parameters click create report to generate a table containing your data.
  • Placeholder - This is a placeholder button for future functions
  • Placeholder - This is a placeholder button for future functions
  • Return Home Button - Click this button to go back to the home screen

Data Table

The data table is generated using the DataTables library. It has multiple features such as pagination, search, and sorting.

Table Features

  • Searching - In the top right is a search bar. Search by key words and the table will pull up the results immediately. No need to press a button.
  • Sorting - Click at the top of a column to order by that column. Click again to switch between descending and ascending order.
  • Pagination - Select different pages in the lower right hand corner. One arrow will advance or go back a single page. Clicking on the double arrows will take you to the first or last page.

Form

The Form is where you will input the database information you wish to connect to. Keep in mind leaving the username and password blank will result in the tool using windows authentication for login.

Database Form Fields

  • Database Engine - Manages data storage and retrieval in a DBMS like MySQL or PostgreSQL.
  • Database Name - Identifier for a specific database instance within the DBMS.
  • Database Host - Server or network location where the database engine and its data are stored.
  • Database Driver - Enables applications to communicate with specific types of databases.
  • Database Username - Credential used to authenticate access to a database.
  • Database Password - Security key required for secure access to a database instance.

Current Connection and Status

In the right pane is the database you are currently connection to. Beneath that is your status/messages box. This is where any status messages will appear to let you know about any errors.

Common Messages

  • Successful Connection - This messages appears when a successful connection is made.
  • Error: 'EngineName' isn't an available database backend - This will appear when an invalid engine is entered
  • Error: '28000',... - This will appear when a database name could not be found
  • Error: '08001',... - This will appear when the database host could not be found
  • Error: 'IM002',... - This will appear when the database driver could not be found
